# [DevSecOps-Pipeline-Project]

## Beskrivning
Detta repository innehåller DevSecOps-Pipeline-Project, en grundläggande implementation av en automatiserad CI-pipeline för en Python-baserad webbapplikation åt "NordTech AB". Projektet demonstrerar hur man bygger, testar och paketerar kod med fokus på säkerhet och automation.
